## 🎯 Overview

This repository contains **enterprise-grade AI-powered workflows** that automate critical Go-To-Market (GTM) operations, from win/loss analysis to ICP segmentation and account intelligence. Built on N8N and powered by Anthropic's Claude AI, these workflows eliminate manual RevOps tasks, surface actionable insights, and drive data-informed GTM decisions.

### Key Capabilities

- 🤖 **AI-Powered Analysis**: Claude AI processes deal data, identifies patterns, and generates strategic recommendations
- 🔄 **End-to-End Automation**: From data extraction to insight delivery, fully automated workflows
- 📊 **Revenue Intelligence**: Real-time deal scoring, win/loss patterns, and ICP refinement
- 🔗 **Native CRM Integration**: Direct HubSpot, Salesforce, and CRM API connections
- 📈 **Scalable Architecture**: Handles thousands of deals, accounts, and data points
- 🔒 **Enterprise Security**: OAuth2, encrypted credentials, audit trails

## 💡 Why AI-Enabled GTM Automation?

### The GTM Operations Challenge

Modern revenue teams face mounting complexity:
- **Data Overload**: Sales teams generate thousands of interactions daily across multiple tools
- **Manual Analysis**: RevOps spends 40%+ of time compiling reports and dashboards
- **Delayed Insights**: By the time patterns emerge manually, opportunities are missed
- **Inconsistent ICP**: Account segmentation based on outdated assumptions, not evidence

### The AI Automation Solution

| Traditional Manual Process | AI-Automated Workflow | Time Saved |
|---------------------------|----------------------|-----------|
| Export 90 days of deal data | ✅ Automated via HubSpot API | 30 min → 0 min |
| Manually categorize wins/losses | ✅ AI pattern recognition | 2 hours → 0 min |
| Build pivot tables and charts | ✅ Automated analysis | 1 hour → 0 min |
| Write executive summary | ✅ Claude AI generates insights | 1 hour → 0 min |
| Distribute to stakeholders | ✅ Auto-post to Slack/Email | 15 min → 0 min |
| **Total Time:** | **4.75 hours** → **5 minutes** | **95% reduction** |

## 🚀 Getting Started

### Prerequisites

- N8N instance (self-hosted or cloud)
- Anthropic API key
- HubSpot OAuth2 credentials
- Slack workspace (optional)

### Quick Start
```bash
# Clone repository
git clone https://github.com/vinaygangidi/n8n-gtm-workflows.git
cd n8n-gtm-workflows

# Configure environment
cp .env.example .env
# Edit .env with your credentials

# Import workflows to N8N
# Via N8N UI: Import from File
```
