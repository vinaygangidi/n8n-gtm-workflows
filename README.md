# n8n GTM Workflows

This repository contains n8n workflows for Go-To-Market automation use cases including account intelligence, ICP analysis, segmentation, and win/loss intelligence.

The repo is designed to serve as the system of record for GTM workflow definitions, documentation, and version history.
